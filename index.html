Second Semester Project: Smart Mirror

The smart mirror is an innovative project that combines a one-way acrylic mirror with a digital display to provide real-time information such as weather, time, calendar updates, and personalized widgets. The project aims to create an affordable yet high-quality interactive mirror.

Project Research & Planning

Currently, we are in the stage of implementing and developing programs to display relevant information on the mirror. Our primary goal is to ensure optimal visibility and responsiveness while maintaining an aesthetically pleasing design.

Current Progress

Week 7-8 (Feb 18 - Feb 24): Software Setup & Understanding the MagicMirror² Framework Setting Up the Raspberry Pi The first step in building our smart mirror is configuring the Raspberry Pi, which acts as the central processing unit of our project. This involves selecting the appropriate model of Raspberry Pi, installing Raspberry Pi OS, and ensuring that it runs optimally. The Raspberry Pi OS is a lightweight Linux distribution designed for ARM-based processors, making it an ideal choice for embedded systems like smart mirrors. After setting up the OS, the next step is to install and configure the MagicMirror² framework. This open-source modular platform enables the integration of various widgets, including time, calendar events, news, and weather updates. We will also ensure that dependencies such as Node.js and npm are correctly installed, as they are crucial for module customization.
