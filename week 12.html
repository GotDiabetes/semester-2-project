🗓️ Week 12 - Post-MVP Reflection & Final Product Planning

✅ Task 1: MVP Feedback Integration & Iteration Planning

🔍 Feedback Review & Analysis

✨ What were the key strengths of your MVP?
Some key strengths in my MVP were the core features, which included displaying the 🌦️ weather, 🎵 Spotify, ⏰ time, and a 🗓️ customized calendar.

🛠️ What areas of improvement did testers or mentors suggest?
Feedback included:
- “Bland, black and white is just a little bit dull and boring.” 🎨
- “It would be awesome to have a real mirror and facial recognition that greets users by name.” 🪞👤

🚧 Were there any unexpected challenges or issues that emerged?
Yes — we had several issues trying to implement the camera/mirror feature 🎥. Even with GitHub modules, we couldn’t integrate it smoothly. We're now planning to use a one-way mirror 🔍🪞.

🧪 How did users interact with your project? Any usability concerns?
Users customized a calendar 🗓️ and displayed Spotify songs 🎧. A minor issue was the camera showing FPS and RAM info ⚙️📊, which we plan to remove.

🛠️ Strategizing Improvements

🚀 Top 3 changes to elevate the project:

  • 🔍 Use one-way reflective glass
  • 📺 Integrate YouTube videos
  • 🎨 Improve color and layout

🧩 Missing features that would improve quality:
Facial recognition 🧠 and voice control 🗣️ would enhance user experience.

⏳ Prioritization based on time and impact:

  • 🎨 UI redesign first (quick, high impact)
  • 🪞 One-way mirror setup (core experience)
  • 📺 YouTube & facial recognition (if time allows)
